# 💳 Credit Card Fraud Detection

A Machine Learning project that identifies fraudulent credit card transactions using data analysis, preprocessing, and classification techniques. This project aims to improve financial security by accurately distinguishing between legitimate and fraudulent transactions.

---

## 📌 Project Overview

Credit card fraud has become one of the most common financial crimes in the digital era. This project uses Machine Learning algorithms to analyze transaction patterns and detect suspicious activities with high accuracy.

The workflow includes:

- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Fraud Prediction

---

## 🚀 Features

✅ Transaction data preprocessing
